                                                               *zavigate-config*
==============================================================================
Configuration management for zavigate.nvim.                    *zavigate.config*

 Merges user configuration with defaults.

 This module is configured automatically by `require("zavigate").setup()`.

Config                                                  *zavigate.config.Config*

    Fields: ~
        {options}  (Config.Options)                      Merged user and default options.
                                                          Configuration management.
        {setup}    (fun(opts?:Config.Options.User):nil)  Merge user options into defaults.


                                                       *zavigate-config-options*
Config.Options                                  *zavigate.config.Config.Options*

    Fields: ~
        {disable_keymaps}  (boolean)  Disable default keymaps. Default: false.
                                       User-facing configuration options.


Config.Options.User                        *zavigate.config.Config.Options.User*

    Fields: ~
        {disable_keymaps?}  (boolean)  Disable default keymaps. Default: false.
                                        User-supplied options (all optional).


Config.Options.Defaults : Config.Options*zavigate.config.Config.Options.Defaults*

    Type: ~
        (Config.Options)


                                                         *zavigate-config-setup*
M.setup({opts?})                                                       *M.setup*
